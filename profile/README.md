  <b>interface + fn</b><br>
  당신과 기술 사이의 접점에서 기능합니다.
</p>

---

**interfn**은 인터페이스를 설계하고, 그 위에서 기능을 구현하는 조직입니다.

약속(interface)만 있고 구현은 없는 상태에서 시작합니다.
하지만 그 약속이 모든 것을 가능하게 합니다.
정의하는 순간, 결과가 나옵니다.

```
interface interfn {
  fn call(input) → output;
}
```
